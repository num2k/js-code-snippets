---

`Math` 객체에는 그래프와 애니메이션 표현을 위한 수학 계산용 메소드가 있습니다.

## 구문
- `Math.abs(값)`: 절댓값을 계산
- `Math.pow(값1, 값2)`: 값1의 값2승을 계산
- `Math.sign(값)`: 값의 부호 계산(음수는 -1, 양수는 1, 0은 0)
- `Math.sqrt(값)`: 제곱근을 계산
- `Math.log(값)`: 자연 로그를 계산
- `Math.exp(값)`: 지수 함수를 계산
- `Math.E`: 자연 로그 e를 반환

## 예시
```js showLineNumbers
Math.abs(-10); // -10의 절댓값은 10
Math.pow(2, 10); // 2의 10승은 1024
Math.sign(2); // 2는 양수이므로 1
Math.sign(-2); // -2는 음수이므로 -1
Math.sqrt(16); // 16의 제곱근은 4
Math.log(Math.E); // 지연로그 e는 1
```
