title: 문자열 검색(부분 검색)
---
대상 문자열에 검색 문자열이 포함되어 있는지 확인 할 때 사용합니다. 결과는 `Boolean` 타입으로 반환 됩니다.

## 구문
- `문자열.includes(검색문자열, [검색시작인덱스])`: 문자열 포함 여부 확인
- `문자열.startsWith(검색문자열, [검색시작인덱스])`: 문자열 시작 문자 확인
- `문자열.endsWith(검색문자열, [검색시작인덱스])`: 문자열 종료 문자 확인

## 예시

```js showLineNumbers
const testString = 'JavaScript 스니펫';

testString.includes('JavaScript'); // true

testString.startsWith('스니펫'); // false

testString.endsWith('스니펫'); // true
```
