## 구문

- `문자열.length`: 문자열의 길이
- `Array.from(문자열).length`: 문자열의 길이

## 예시

```js
'javascript'.length; // 10
```
이모지는 문자열의 길이를 `length`로 확인하면 부정확 합니다.
```js
'🚀'.length; // 2
'🚨'.length; // 2
```
자바스크립트는 기본적으로 하나의 문자를 2바이트로 표현하지만, 이모티콘과 특수 문자는 4바이트로 표현합니다. 이렇게 4바이트 표현되는 문자를 서로게이트 쌍(Surrogate Pair)이라고 합니다. <br />서로게이트 쌍을 하나의 문자로 인식하기 위해서는 `Array.from()`을 사용합니다.
```js
Array.from('🚀').length; // 1
Array.from('🚨').length; // 1
```
