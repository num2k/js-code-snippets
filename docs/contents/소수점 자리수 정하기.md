숫자나 소수점의 자릿수를 지정할 때 사용합니다.

## 구문
- `숫자값.toFixed(자릿수)`: 소수점 이하 자릿수 지정
- `숫자값.toPrecision(자릿수)`: 자릿수 지정 정밀도 반환

`toFixed()`는 지정된 소수점 자릿수로 변환후 문자열로 반환됩니다.

```js showLineNumbers
(0.33333).toFixed(2); // '0.33'
(123.5678).toFixed(1); // '123.6' (근사치로 반환)
(1.2).toFixed(4); // '1.2000' (소수점이하 남은 자리수 만큼 0이 채워짐)
```

`toPrecision()`은 들어온 값을 지정한 자릿수의 정밀도로 변환하여 문자열로 반환됩니다.
```js showLineNumbers
(0.33333).toPrecision(2); // '0.33' (정밀도2)
(123.456).toPrecision(3); // '123' (정밀도3)
(4.56).toPrecision(2); // '4.6' (근사치로 반환)
(10).toPrecision(4); // '10.00' (정밀도 4가 되도록 0으로 채움)
```
